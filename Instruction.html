Instructions



  1. Players have to login to the game using the password provided to them
  2. There will be a total of 4 rounds in the game
  3. Each round can have 1-4 questions/puzzles
  4. After solving each question, follow the hints/instructions that manipulate your answer to give a code + a numeric value
  5. Google the code to get the name of a money heist character
  6. The final answer in all rounds is a name + a numeric value
  7. Remember all your answers as they contain the secret to reach the end of the heist
  8. Each team gets 2 chances per answer (irrespective of the no. of devices you are logging in through)
  9. After the 4 rounds follow the given instruction to end the heist
